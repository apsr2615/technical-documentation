INTRODUCTION

Python is a versatile, high-level programming language known for its readability and simplicity. DEVOLOPED BY GUIDO VAN ROSSUM and first released in 1991, Python is designed to emphasize code readability and developer productivity. Its straightforward syntax makes it beginner-friendly, while its powerful libraries and frameworks make it a top choice for professionals in various fields.

Hello World!

The first program usually starts with printing "Hello, World!" on the screen.

print("Hello, World!")

Variables and Data Types

Python supports various data types, including integers, floats, strings, and booleans.

# Integer x = 10
# Float y = 3.14
# String name = "Alice"
# Boolean is_active = True

Basic Operators

  • Arithmetic Operators: +, -, *, /, ** (power), % (modulus).
  • Comparison Operators: ==, !=, <, >, <=, >=.
  • Logical Operators: and, or, not.
a = 5
b = 3
# Arithmetic
sum_ab = a + b # 8
power = a ** b # 125
# Comparison
is_equal = (a == b) # False
# Logical
result = (a > b) and (b < 10) # True

Control Structures

Conditional Statements

age = 18
if age >= 18:
print("You are an adult.")
elif age >= 13:
print("You are a teenager.")
else:
print("You are a child.")

loops

  • For Loop: Iterate over a sequence.
  • While Loop: Repeat as long as a condition is true.
# For loop
for i in range(5):
print(i) # Prints 0 to 4
# While loop
count = 0
while count < 5:
print(count)
count += 1

Functions

Functions help you encapsulate code for reuse.

def greet(name):
return f"Hello, {name}!"
print(greet("Alice")) # Output: Hello, Alice!
Lists Tuples and Dictionaries

Lists:

A list is an ordered collection that is mutable.

fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits[1]) # Output: banana

Tuples:

A tuple is an ordered, immutable collection.

point = (3, 5)
print(point[0]) # Output: 3

Dictionaries:

A dictionary is a collection of key-value pairs

person = {"name": "Alice", "age": 25}
print(person["name"]) # Output: Alice

Exception Handling

Use try, except to handle errors gracefully.

try:
result = 10 / 0
except ZeroDivisionError:
print("Cannot divide by zero!")

File Handling

You can read from and write to files.

# Writing to a file with open("example.txt", "w") as f:
f.write("Hello, world!")
# Reading from a file
with open("example.txt", "r") as f:
content = f.read()
print(content)
